# Dorothy User Configuration

This is my user configuration for the [Dorothy](https://github.com/bevry/dorothy) dotfile ecosystem.

## Installation

Pre-requisites:
```bash
sudo apt-get update
sudo apt-get install bash curl
```

Install Dorothy:
```bash
bash -ic "$(curl -fsSL https://dorothy.bevry.me/install)"
```

Verify installation:
```bash
dorothy theme
```
